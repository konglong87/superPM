## 执行流程

### 步骤 1: 读取前置数据

**如果有需求调研报告**：

使用 Read 工具读取 `docs/01-需求调研/需求调研报告.md`

提取：
- 产品名称
- 行业赛道
- 目标用户
- 核心痛点

**如果没有需求调研报告**：

进入快速模式，使用 AskUserQuestion 收集基础信息：

> 📝 快速模式 - 请提供产品信息：
>
> 问题1：产品名称是什么？
>
> 问题2：行业赛道是什么？
>
> 问题3：目标用户是谁？

---

### 步骤 2: 确定搜索方向

使用 AskUserQuestion 询问：

> 🎯 我要重点分析哪些方面？
>
> A) 市场规模与增长趋势 - 市场有多大、增长速度如何
> B) 竞品分析 - 主要竞品有哪些、优劣势对比
> C) 用户需求与行为 - 用户偏好、消费习惯
> D) 行业政策与风险 - 政策法规、合规要求
> E) 全部分析（推荐但耗时较长）

用户选择后，记录到变量 `MARKET_FOCUS`

---

### 步骤 3: 执行联网搜索

**关键原则**：
- MCP Search 优先（如 exa-mcp）
- WebSearch 降级
- 明确数据源优先级

---

#### 3.1 尝试 MCP Search

**如果项目配置了 exa-mcp**：

使用 exa_search 工具搜索：

```
工具: exa_search
参数:
  - query: "{产品名称} {行业赛道} 市场规模 2024 2025"
  - includeDomains: ["iresearch.cn", "analysys.cn", "questmobile.com.cn", "iimedia.cn"]
  - numResults: 5
```

**数据源优先级**（所有搜索通用）：

1. **权威报告**：
   - 艾瑞咨询（iresearch.cn）
   - 易观分析（analysys.cn）
   - QuestMobile（questmobile.com.cn）
   - 艾媒咨询（iimedia.cn）

2. **应用数据**：
   - 七麦数据（qimai.cn）
   - 蝉妈妈（chanmama.com）

3. **行业媒体**：
   - 36氪（36kr.com）
   - 虎嗅（huxiu.com）

4. **政策法规**：
   - 工信部（miit.gov.cn）
   - 网信办（cac.gov.cn）

---

#### 3.2 降级 WebSearch

**如果 MCP 不可用**：

使用 WebSearch 工具：

**搜索关键词组合**：

如果选择"市场规模"：
```
"{行业赛道} 市场规模 2025 site:iresearch.cn"
"{行业赛道} 行业报告 2025 site:analysys.cn"
"{行业赛道} 市场增长趋势"
```

如果选择"竞品分析"：
```
"{行业赛道} 竞品分析"
"{行业赛道} 主要玩家 排名"
"{产品类型} 对比评测"
```

如果选择"用户需求"：
```
"{目标用户} 消费行为 2025"
"{行业赛道} 用户画像 报告"
"{行业赛道} 用户需求 调研"
```

如果选择"行业政策"：
```
"{行业赛道} 政策法规 2025 site:miit.gov.cn"
"{行业赛道} 合规要求"
```

---

### 步骤 4: 分析搜索结果

AI 整理搜索结果，提取关键信息：

**市场规模分析**：
- 市场规模（金额、用户数）
- 年增长率
- 细分市场占比

**竞品分析**：
- 主要竞品名称
- 竞品市场份额
- 竞品优劣势

**用户需求分析**：
- 用户偏好
- 消费习惯
- 痛点分布

**行业政策分析**：
- 相关政策法规
- 合规要求
- 行业风险

## 兜底机制

### 场景 1: MCP Search 不可用

自动降级使用 WebSearch。

提示：
> ⚠️ MCP Search 不可用，已切换到 WebSearch
>
> 数据来源可能不够权威，建议交叉验证

### 场景 2: 搜索结果不足

如果搜索结果不足：

> ⚠️ 搜索结果较少，无法全面分析
>
> 您可以选择：
> A) 调整搜索关键词重试
> B) 手动输入市场数据
> C) 跳过部分分析维度

### 场景 3: 无需求数据

提供快速模式，允许手动输入基础信息。

---

## 注意事项

1. **MCP优先**：优先使用MCP Search获取权威数据
2. **数据源优先级**：权威报告 > 应用数据 > 行业媒体 > 政策法规
3. **数据标注来源**：所有数据必须标注来源，便于验证
4. **交叉验证**：重要数据建议多个来源交叉验证
5. **Markdown存储**：市场报告人类可读可编辑
